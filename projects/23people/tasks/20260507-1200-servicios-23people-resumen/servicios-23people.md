# Servicios de 23people — Resumen Rápido

23people SpA es una empresa chilena de tecnología fundada en 2014. Ofrece **cuatro servicios especializados** para la evolución digital empresarial:

---

## 1. 🧑‍💻 Tech Talent

Formación, selección y gestión de equipos tecnológicos.

| Modalidad | Descripción |
|---|---|
| **Outsourcing Tech** | Subcontratación de equipos bajo la estructura legal de 23people (largo plazo, 6+ meses) |
| **Hunting Tech** | Búsqueda y selección para contratación directa del cliente |
| **Alta Especialización** | Proyectos cortos (< 6 meses) con el equipo interno de Ingeniería |

**Diferenciadores clave:**

- Rotación voluntaria < 20% anual
- Permanencia promedio > 1.6 años por proyecto
- Alineación horaria Chile/LATAM → US
- Altos NPS de clientes y profesionales

---

## 2. 🔗 Integrations

Conexión de sistemas vía APIs, eventos y automatizaciones.

- Integraciones financieras y de scoring
- Apps satélite para ERPs (SAP, Oracle)
- Automatización de flujos cross-system
- Pipelines de datos para analítica y AI

**Stack:** RabbitMQ, Kafka, AWS SQS/SNS, n8n, Temporal.io, Node.js/TypeScript, Python, Go, Airflow, dbt, Docker/Kubernetes, Terraform

---

## 3. ☁️ Cloud Optimization

Minimización de costos cloud sin sacrificar rendimiento ni seguridad.

- Migraciones y modernizaciones cloud
- Arquitecturas híbridas cloud + on-premise
- Conectores seguros y API Gateways

---

## 4. 📊 Data & AI

Ingeniería de datos para decisiones y soluciones basadas en IA.

- Pipelines de datos: CRM, ERP, transaccional → Snowflake, BigQuery, Redshift
- Pipelines documentales para RAG y agentes de IA
- Embeddings, búsqueda vectorial, LangChain, LlamaIndex
